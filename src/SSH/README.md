# eYogi Gurukul Platform

A comprehensive educational platform for spiritual learning and gurukul management.

## Getting Started

```bash
npm install
npm run dev
```

## Features

- Student and teacher dashboards
- Course management
- Enrollment system
- Certificate generation
- AI-powered chatbot
- SEO optimization
- Admin panel

## Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS
- Supabase
- React Router
